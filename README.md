# 🦇 Alfred Dashboard

Telegram Web App панель управления для персонального ассистента Альфреда.

## Особенности

- Стиль старых компьютерных игр (CRT, зелёный монитор)
- Адаптивный дизайн для мобильных устройств
- Интеграция с Telegram Web App
- Автоматическое обновление времени
- Уведомления и статусы

## Использование

1. Откройте в Telegram через Web App
2. Нажмите "ОБНОВИТЬ ДАННЫЕ" для синхронизации
3. Используйте "ЗАКРЫТЬ" для выхода из Web App

## Технологии

- HTML5, CSS3, JavaScript
- Telegram Web App API
- GitHub Pages для хостинга

## Разработка

Для локальной разработки:
1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для тестирования Telegram Web App используйте [@BotFather](https://t.me/botfather) для создания тестового бота
